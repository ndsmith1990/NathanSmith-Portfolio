As promised, I have been working on World of Health intermittently when I am not working at my current job or studying for finals, which means that it has been slow going. Functionality

Currently, the graph element is fully functional only on the weight and BMI trackers. My idea is to get full functionality on a few trackers at a time. This should make it fairly easy to get all the trackers done. I am making sure that my trackers' functions are not specific to that tracker page, but for at least one of the pages I will have to make each piece of code specific to that tracker, simply because that tracker is different from all the others.

Source: hackernoon.com

Moving Forward

Alright, so the weight graph is done, so what is next? The next piece of functionality that I plan to work on is the table on the weight tracker. Implementing this feature should get the all the trackers except the "other" tracker up and running. So the next blog post about this application should announce that the fitness side of the app is fully functional with the exception of the other tracker.
